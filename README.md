# Global Pollution Analysis & Energy Recovery Prediction (Prototype)

## Project Overview

This project is my **first Machine Learning project** and is being developed as a learning-based prototype. The goal is to analyze global pollution data, understand the relationship between different pollution indicators, and predict the amount of energy that can be recovered from pollution and industrial waste using Machine Learning.

This project is not intended to be a production-ready application. Instead, it documents my learning journey while applying data analysis, visualization, regression models, and eventually a full-stack web application.

---

## Project Objectives

* Analyze global pollution data.
* Perform data cleaning and preprocessing.
* Explore the dataset using data visualization.
* Train and compare different Machine Learning regression models.
* Predict Energy Recovery (in GWh) based on pollution-related features.
* Extend the project into a web application with a frontend and backend.

---

## Dataset

The dataset contains information related to:

* Country
* Year
* Air Pollution Index
* Water Pollution Index
* Soil Pollution Index
* CO₂ Emissions
* Industrial Waste
* Renewable Energy Percentage
* Population
* GDP per Capita
* Energy Recovery (Target Variable)
